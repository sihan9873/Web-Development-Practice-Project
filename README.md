# 星辉科技企业官网

## 项目简介

这是一个完整的企业官网系统，包含前端静态网站和后端 API 服务，支持多用户账号管理和数据持久化存储。

### 功能页面

- **首页** (`index.html`) - 企业主页，展示核心能力与价值主张
- **企业介绍** (`about.html`) - 公司概况、发展历程、企业文化
- **招聘信息** (`recruit.html`) - 职位发布与在线投递简历
- **联系我们** (`contact.html`) - 联系方式与在线留言表单
- **留言板** (`message.html`) - 访客留言功能
- **用户登录** (`login.html`) - 用户登录页面
- **用户注册** (`register.html`) - 用户注册页面
- **管理后台** (`admin.html`) - 简历与留言管理后台（需管理员权限）

## 🆕 多用户账号系统

### 核心功能

✅ **用户注册与登录**
- 用户可以通过邮箱注册账号
- 支持 JWT Token 认证
- 密码加密存储（bcrypt）

✅ **多用户数据隔离**
- 每个用户有独立的账号
- 用户可以查看和管理自己的投递记录
- 支持用户信息管理

✅ **管理员后台**
- 管理员可以查看所有用户的投递记录
- 支持简历状态管理（待审核/审核中/已通过/已拒绝）
- 用户管理功能
- 数据统计与分析

✅ **数据持久化**
- 使用 MongoDB 数据库存储
- 支持本地 MongoDB 或 MongoDB Atlas 云数据库
- 数据永久保存，支持跨设备访问

## 两种运行模式

### 模式一：纯前端模式（localStorage）

无需后端，直接在浏览器中打开即可使用。

**特点：**
- ✅ 快速启动，无需配置
- ❌ 数据仅保存在本地浏览器
- ❌ 无法跨设备访问
- ❌ 不同用户数据隔离

**使用方法：**
```bash
# 直接在浏览器打开 index.html 或使用本地服务器
python -m http.server 8000
```

### 模式二：完整后端模式（推荐）

需要 Node.js 和 MongoDB，支持多用户共享数据。

**特点：**
- ✅ 数据存储在数据库，永久保存
- ✅ 支持多用户账号系统
- ✅ 管理员后台可查看所有数据
- ✅ 支持跨设备访问
- ✅ 简历状态管理

**快速开始：**
```bash
# 1. 安装依赖
npm install

# 2. 配置环境变量（复制 env.example 为 .env）
cp env.example .env

# 3. 启动 MongoDB（如果使用本地数据库）

# 4. 启动后端服务器
npm start

# 5. 启动前端（使用 Live Server 或其他静态服务器）
```

详细部署说明请查看 **[BACKEND_SETUP.md](./BACKEND_SETUP.md)**

## 功能说明

### 简历投递功能

- 用户可以在招聘页面填写个人信息并投递简历
- 投递记录保存在浏览器本地存储（localStorage）中
- 用户可以在投递页面查看自己的投递记录

### 管理后台功能

访问 `admin.html` 可以查看和管理所有投递记录。

**主要功能：**

1. **数据统计**
   - 总投递数
   - 当前显示数量
   - 职位类型数量
   - 今日投递数

2. **搜索与筛选**
   - 按姓名、邮箱、电话、职位搜索
   - 按职位类型筛选
   - 按时间或姓名排序

3. **数据管理**
   - 查看简历详情
   - 删除投递记录
   - 导出 JSON 格式数据
   - 导出 CSV 格式数据（支持 Excel 打开）
   - 清空所有数据

## 使用说明

### 本地运行

1. 直接在浏览器中打开 `index.html` 即可
2. 或者使用本地服务器：
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js (需要安装 http-server)
   npx http-server
   ```

### 访问管理后台

1. 打开 `admin.html` 文件
2. 或者在首页底部点击"管理后台"链接
3. 页面会自动加载浏览器中存储的所有投递记录

### 导出数据

1. 在管理后台页面，点击"导出 JSON"或"导出 CSV"按钮
2. 文件会自动下载到本地
3. JSON 格式适合程序处理，CSV 格式可以用 Excel 打开
